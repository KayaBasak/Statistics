# üìä A Gentle Introduction to Probability & Statistics

This repository contains my notes, Google Colab notebooks, and supporting materials for learning and revisiting **probability and statistics** from an intuition-first perspective.

The goal of this repository is not to be overly mathematical, but to build a **clear mental model** for key statistical ideas using explanations, simulations, and visualizations in Python.

- Probability basics  
- Random experiments and sample spaces  
- Random variables and distributions  
- PMF, PDF, and CDF  
- The normal distribution and the empirical rule  
- Sampling variability  
- Law of Large Numbers  
- Population vs sample  

- Hypothesis testing framework
- Null and alternative hypotheses
- Test statistics (z- and t-statistics)
- p-values and their interpretation
